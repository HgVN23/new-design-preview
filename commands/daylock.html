/daylock

Công dụng

Khóa hoặc mở khóa thời điểm trời luôn sáng

Đối tượng có thể dùng Command

Player có OP

Chủ Map

Addon

NPC /execute /function /gametest Command Block

Cú pháp

/daylock [lock: Boolean]

[lock: Boolean] là phần trạng thái

Danh sách:

true là bật

false là tắt

Mặc định là true nếu bỏ qua thành phần này

Lưu ý

/daylock false thì /gamerule dodaylightcycle sẽ thành true cho dù trước đó bạn để false
