/gamerule

Công dụng

Điều chỉnh luật lệ của World

Đối tượng có thể dùng Command

Player có OP

Chủ Map

Addon

NPC /execute /function /gametest Command Block

Cú pháp

/gamerule <rule: BoolGameRule> [value: Boolean]

<rule: BoolGameRule> là id mang tính bật hoặc tắt Danh sách
[value: Boolean] là trạng thái

Danh sách

true là bật

false là tắt

Nếu bạn bỏ qua thành phần này thì Command sẽ thông báo trạng thái của gamerule đang bật hoặc tắt

/gamerule pvp false

Player sẽ không đánh được nhau

/gamerule <rule: IntGameRule> [value: int]

<rule: IntGameRule> là id mang tính dữ liệu số Danh sách
[value: int] là dữ liệu số

Nếu bạn bỏ qua thành phần này thì Command sẽ thông báo trạng thái của gamerule đang bật hoặc tắt

/gamerule randomtickspeed 5

Tốc độ tự nhiên phát triển tăng lên 5

Lưu ý

Giới hạn [value: int]

functioncommandlimit

Từ bằng 0 đến bằng 10000

maxcommandchainlength

Từ bằng 0 đến bằng 2147483647

randomtickspeed

Từ bằng 0 đến bằng 4096

spawnradius

Từ bằng 0 đến bằng 128
