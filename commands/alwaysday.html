Đối tượng có thể dùng Command

Command Block

Cú pháp

/alwaysday [lock: Boolean]

[lock: Boolean] là phần trạng thái

Danh sách:

true là bật

false là tắt

Mặc định là true nếu bỏ qua thành phần này

Lưu ý

/alwaysday false thì /gamerule dodaylightcycle sẽ thành true cho dù trước đó bạn để false
