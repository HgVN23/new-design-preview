/setblock

Công dụng

Đặt 1 Block

Đối tượng có thể dùng Command

Player có OP

Chủ Map

Addon

NPC /execute /function /gametest Command Block

Cú pháp

/setblock <position: x y z> <tileName: Block> [blockStates: block states] [replace|destroy|keep]

<position: x y z> là tọa độ

<tileName: Block> là id Block

[blockStates: block states] là trạng thái Block Danh sách

Mặc định là 0 nếu bỏ qua thành phần này

[replace|destroy|keep] là thao tác

Danh sách

replace là thay thế (không rơi ra Block)

destroy là phá hủy (rơi ra Block)

keep là giữ nguyên

Mặc định là replace nếu bỏ qua thành phần này

/setblock 1 1 1 wool ["color":14] replace

Đặt len đỏ vào 1 1 1 thay thế Block cũ ở đó

/setblock <position: x y z> <tileName: Block> [tileData: int] [replace|destroy|keep]

<position: x y z> là tọa độ

<tileName: Block> là id Block

[tileData: int] là dữ liệu Block

Mặc định là 0 nếu bỏ qua thành phần này

[replace|destroy|keep] là thao tác

Danh sách

replace là thay thế (không rơi ra Block)

destroy là phá hủy (rơi ra Block)

keep là giữ nguyên

Mặc định là replace nếu bỏ qua thành phần này

/setblock 1 1 1 wool 14 replace

Đặt len đỏ vào 1 1 1 thay thế Block cũ ở đó
