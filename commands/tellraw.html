/tellraw

Công dụng

Gửi Chat dưới dạng mã tới mục tiêu xác định

Đối tượng có thể dùng Command

Player có OP

Chủ Map

Addon

NPC /execute /function /gametest Command Block

Cú pháp

/tellraw <target: target> <raw json message: json>

<target: target> là mục tiêu

Chỉ được chọn mục tiêu là Player

<raw json message: json> là tin nhắn viết dưới dạng {"rawtext": [...]} Danh sách
/tellraw Hg {"rawtext": [{"text": "Hi"}]}

Hg nhận được Chat Hi
