/say

Công dụng

Khiến đối tượng sử dụng Command này gửi Chat đã ghi trong Command

Đối tượng có thể dùng Command

Player có OP

Chủ Map

Addon

NPC /execute /function /gametest Command Block

Cú pháp

/say <message: message>

<message: message> là đoạn Chat

/say Hi

Đối tượng sử dụng sẽ Chat là Hi

Ví dụ Hg sử dụng thì Chat sẽ hiện là [Hg] Hi

Lưu ý

Hover Note của Command Block không ghi gì thì đầu ra sẽ là [!]. Nếu Hover Note có ghi Test thì đầu ra sẽ là [Test]

Khi sử dụng Command này tất cả Player đều có thể thấy được Chat
