/structure

Công dụng

Thay thế cho Structure Block, để Save hoặc Load cấu trúc

Đối tượng có thể dùng Command

Player có OP

Chủ Map

Addon

NPC /execute /function /gametest Command Block

Cú pháp

/structure delete <name: string>

delete là xóa

<name: string> là id cấu trúc

/structure delete wiki:hg

Xóa cấu trúc wiki:hg

/structure load <name: string> <to: x y z> [rotation: Rotation] [mirror: Mirror] [animationMode: StructureAnimationMode] [animationSeconds: float] [includesEntites: Boolean] [includesBlocks: Boolean] [integrity: float] [seed: string]

load là hiện

<name: string> là id cấu trúc

<to: x y z> là tọa độ hiện

[rotation: Rotation] là xoay

Danh sách

0_degrees là 0°

90_degrees là 90°

180_degrees là 180°

270_degrees là 270°

Mặc định là 0_degrees nếu bỏ qua thành phần này

[mirror: Mirror] là đối xứng

Danh sách

none là không

x là trục tọa độ x

xz là trục tọa độ xz

z là trục tọa độ z

Mặc định là none nếu bỏ qua thành phần này

[animationMode: StructureAnimationMode] là hoạt ảnh cấu trúc xuất hiện

Danh sách

block_by_block là từng Block

layer_by_layer là từng tằng

Mặc định là không có nếu bỏ qua thành phần này

[animationSeconds: float] là khoảng thời gian Delay của hoạt ảnh cấu trúc xuất hiện

Mặc định là 0.00 nếu bỏ qua thành phần này

[includesEntites: Boolean] là chứa Entity

Danh sách

true là bật

false là tắt

Mặc định là true nếu bỏ qua thành phần này

[includesBlocks: Boolean] là chứa Block

Danh sách

true là bật

false là tắt

Mặc định là true nếu bỏ qua thành phần này

[integrity: float] là tỉ lệ load được hết cấu trúc

Giới hạn từ bằng 0.00 đến bằng 1

Mặc định là 1 nếu bỏ qua thành phần này

[seed: string] cần 1 ai đó giải thích hộ

/structure load wiki:hg 1 1 1 0_degrees none block_by_block 5 false true 1

Hiện cấu trúc wiki:hg từ tọa độ 1 1 1, không xoay, không đối xứng, hiện từng Block delay 5 giây, không có Entity, có Block và 100% Load hết

/structure save <name: string> <from: x y z> <to: x y z> [includesEntites: Boolean] [saveMode: StructureSaveMode] [includesBlocks: Boolean]

save là lưu

<name: string> là id cấu trúc

<from: x y z> là tọa độ bắt đầu

<to: x y z> là tọa độ kết thúc

[includesEntites: Boolean] là chứa Entity

Danh sách

true là bật

false là tắt

Mặc định là true nếu bỏ qua thành phần này

[saveMode: StructureSaveMode] là cách lưu

Danh sách

disk là lưu vĩnh viễn

Mở lại World không mất

memory là lưu tạm thời

Mở lại World sẽ mất

Mặc định là memory nếu bỏ qua thành phần này

[includesBlocks: Boolean] là chứa Block

Danh sách

true là bật

false là tắt

Mặc định là true nếu bỏ qua thành phần này

/structure save wiki:hg 0 0 0 2 2 2 false disk true

Lưu cấu trúc trong id wiki:hg từ 0 0 0 đến 2 2 2, không có Entity, lưu trong Disk và có Block
