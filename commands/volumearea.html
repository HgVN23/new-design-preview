/volumearea

Công dụng

Giới hạn vùng Fog qua tệp tin Volume

Đối tượng có thể dùng Command

Player có OP

Chủ Map

Addon

NPC /execute /function /gametest Command Block

Cú pháp

/volumearea add <identifier: string> <from: x y z> <to: x y z> [name: string]

add là tạo

<identifier: string> là id File volumearea bạn tạo qua Addon

<from: x y z> là tọa độ bắt đầu

<to: x y z> là tọa độ kết thúc

[name: string] là đặt id cho volumearea

Mặc định là Volume + i với i là số từ 0 đến 99 nếu bỏ qua thành phần này

/volumearea add custom:test 0 0 0 0 0 0 test

Tạo giới hạn Fog custom:test dưới id là test trong khoảng 0 0 0 đến 0 0 0

/volumearea list [all-dimensions: VolumeAreaAllDimensions]

list là danh sách volumearea

[all-dimensions: VolumeAreaAllDimensions] là tất cả chiều không gian

Mặc định là chiều không gian bạn đang đứng nếu bỏ qua thành phần này

/volumearea remove <name: string>

remove là bỏ

<name: string> là id của volumearea đã tạo

/volumearea remove test

Bỏ test

/volumearea remove <position: x y z>

remove là bỏ

<position: x y z> là tọa độ volumearea đã tạo

/volumearea remove 0 0 0

Bỏ tất cả volumearea chứa tọa độ 0 0 0

/volumearea remove_all

remove_all là bỏ tất cả

Hướng dẫn tạo File Volume cơ bản

Trước tiên ấn Download để tải Pack mẫu về, sau đó giải nén và chuyển thư mục đó vào behavior_packs

B1: Mở thư mục volumes

B2: Nhân bản tệp tin template.json và đổi tên bất kì (không đổi tên cũng được) hoặc sửa trực tiếp tệp đó luôn

B3: Mở tệp tin vừa tạo, sửa dòng 5 là id của volume và dòng 9 là id của fog. Sau khi xong nhớ ấn lưu

Danh sách id fog hoặc sử dụng id fog do bạn tạo cũng được
