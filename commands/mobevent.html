/mobevent

Công dụng

Điều khiển Event lớn

Đối tượng có thể dùng Command

Player có OP

Chủ Map

Addon

NPC /execute /function /gametest Command Block

Cú pháp

/mobevent <event: MobEvent> [value: Boolean]

<event: MobEvent> là id Event lớn

Danh sách

events_enabled là Event lớn hoạt động

minecraft:ender_dragon_event là Event hồi sinh Ender Dragon

minecraft:pillager_patrols_event là Event Raid khi Player có Bad Omen gần Village

minecraft:wandering_trader_event là Event Spawn Wandering Trader theo đợt

[value: Boolean] là trạng thái

Danh sách

true là bật

false là tắt

Nếu bạn bỏ qua thành phần này thì Command sẽ thông báo trạng thái đang bật hoặc tắt

/mobevent events_enabled false

Mọi Event lớn sẽ bị tắt

Lưu ý

Nếu events_enabled là false thì các event còn lại sẽ không hoạt động kể cả có để là true
